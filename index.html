PROJECTS

Budget Manager A group project made during Purdue's BoilerMake Hackathon. We wanted to create a tool for students to effectivly manage thier finances. This application allows users to track their monthly spending. It also creates personalized spending goals so that users can save money to buy items that they add to their "wishlist".

Panic Bomber A group project made during Purdue's Hello World Hackathon. A two-player game where players compete to pass a bomb to one another to avoid getting blown up when the timer ticks down.

Rugby Data Analysis A group project made during Purdue's Datafest. We were given training and game data from a Canadian Rugby team and used the data to calculate a statndardized wellness factor for each player. We used this to find a corralation between wellness factor and game performace.

Heart Disease Predictor Used a Cleveland heart disease dataset to train a deep learning model. The model predicts with 85% accurarcy if someone has Heart Disease using factors such as age, blood pressure, and heart rate.

Restaurant Success predictor A deep leearning model that uses a dataset from Yelp to predict the success of a new restaurant based on location, type of cuisine, pricing, etc. Work in Progress. This app will allow entrepreneurs to find the optimal location for a new restaurant.

Tic-Tac-Toe AI An unbeatable tic-tac-toe AI built using the min-max algorithm. This was a project I built over the summer when I first started getting into Python and machine learning.
